 ከሰምጥ ሰለቆ ከተሞች ዉስጥ ወደ አዳማ መጉዝ ወደድን:: አዳማ ከተማ በፈጣን አድገት ላይ ካሉ ከተሞች በገምባር ከደምትነት ተጠቀሳለች :: በዉስጡዋ ካሉ አያሌ ተቁማት መካከል የሰለጠነ የሰው ኃይል በማፍራት ላይ የገኛል የአዳማ ሳይስን አና ቴክኖሎጂ ዩኒቨርሲቲ:: አንዴ አውሮፓወያን አቆጣጠር በ 1993 የናዝሬት ቴክኒካል ኮሌጅ ተብሎ አንደተመሰረተ ከታሪክ ማህደሩ መረዳት ችለኛል :: በዚህ ጊዜ አያሌ የአገር ባለሞያዎች አፍረቷል :: በየጊዜው የተለያዩ ማሻሻያዎች አየተደረገለት ለአገሪቱ የተማረ የሰው ሃየልን አያፈራ ተልቅ ሚና አየተጫወተ የገኛል :: በ 2003 ናዝሬት የቴክኒክ መምሀራን ተምርት ኮሌጅ ተብሎ በቁ መምህራነን ማፍራት ጀመረ :: ግቢው ለመማር አና ማስተማር ሚቹ የሆነ አከባቢ አና ነፋሻማ አየር አለው :: የተተከሉት ዛፎች አና የተሰሩት የልማት ስራዎች ግቢውን ውብ አድርገውታል :: ሌላኛው ቀልባችንን የሣበው ነገር ደግሞ በመውጫ በሮች ላይ የሚገኘው የተማሪዎች መቆጣጠሪያ ቴክኖሎጂ ነው :: ተማሪዎች መታወቂያቸው ላይ ባለው ባር ኮድ አማካኝነት በቀላሉ ማንነታቸውን ከነ ፎቶግራፋቸው ማወቅ የቻላል :: እሄም የደህንነት ሰራውን ያቀላትፈዋል:: የአገራችነን የተክኖሎጂ ዓቅም ሽግግር ያከናውናሉ ተብለው ከተቁዋቁዋሙት ተቁማት መሃከል የህ ተቁአም አንዱ ነው :: አዲስ ገቢ ተማሪዎች ለአንድ ዓመት የመጀምሪያ ዓመት ዎይም የfresh course የወስዳሉ :: ከዚያ ደግሞ school መርጠው ለ አንድ ሰሜስተር ያክል ተምረው ሲጨርሱ ደፓርትመንት ይመርጣሉ :: በትምርት ቅበላቸው ጎበዝ ለሆኑት ተማሪዎች ደግሞ የተለያዩ ተጨማሪ ፕሮግራሞች አንዳሉ ሰምተናል :: አነዚህም fact track አና dual ፕሮግራም ይባላሉ :: ዩኒቨርሲቲው በአራት ትምርት ቤቶች የተደራጀ ነው :: አነዚህም 1- አፕሌድ የተፈጥሮ ሳይንስ ትምርት ቤት 2- ሲቪል መህድስና አና አርክቴክቸር ትምርት ቤት 3- የ ኤሌክትሪካል ምህድስና አና ኮምፕዩትንግ ትምርት ቤት አና 4- መካኒካል ፣ ኬሚካል አና ማቴሪያል ሳይንስ ትምርት ቤት ነቸው :: ባጠቃላይ አዳማ ሳይንስ አና ቴክኖሎጂ አያከናወነ ያለው ነቁ አና በቴክኖሎጂ የታነፀ ተውለድን የማፍራት ሰራው ይበል ምያስብል አና ተጠናክሮ መቀጠል አለበት ::
